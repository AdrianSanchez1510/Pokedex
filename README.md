# Pokedex
A flutter application design to act as a pokedex using a Rest-Api 

![Pokedex App](https://i.imgur.com/842Uzo0.png)

# How to use
1. Download flutter
   
2. Use the following commands
- flutter pub get
- flutter run (for android)
- flutter run -d chrome (for web)
- flutter run -d windows

For automated testing use flutter test

3. Use the index or the search bar to look for your favorite pokemon
